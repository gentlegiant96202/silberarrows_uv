🔍 Consignment Agreement Preview Tool

📝 How to Use This Preview Tool:

  1. Select which agreement template you want to preview from the dropdown
  2. The template will load with sample data in the iframe below
  3. Make changes to the HTML file in: renderer/public/templates/[template-name].html
  4. Refresh this page to see your changes instantly
  5. Once happy with your changes, deploy to see them in production

💡 Tip: Keep this page open while editing the templates for instant preview!
